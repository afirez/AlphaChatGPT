巅峰之作

小米SU 7 Ultra是小米汽车推出的高性能纯电动轿车，定位为巅峰性能科技轿车，官方宣称其为"可街可赛"的四门赛车。2025年2月27日正式上市，起售价52.99万元。

作为小米汽车的旗舰产品，SU 7 Ultra融合了极致性能、豪华体验和先进科技，目标是成为地表最快四门量产车。

预售开启仅10分钟，小订量就突破3680台，市场反响热烈。小米SU 7 Ultra的目标定位是"性能比肩保时捷、科技紧追特斯拉、豪华媲美BBA"，全年销售目标为1万台，相当于特斯拉Model S Plaid、宝马M系列和保时捷Taycan全年在华销量的两倍。

技术创新

三电机系统

搭载小米自研的超级电机，组成三电机四驱系统，整体峰值功率强大。

高性能电池

采用宁德时代提供的高性能电池，放电功率可达1300千瓦以上，支持超快充电。

轻量化技术

大量使用碳纤维材料，如1.7m²的超大面积碳纤维车顶，带来12kg减重。

空气动力学

全新的空气动力学设计，提供更稳定的高速驾驶体验。

赛道表现

纽博格林北环赛道

以6分46秒874的官方认证圈速成绩，成为全球最速四门车。2025年5月，雷军透露小米SU7 Ultra量产版正在再次挑战纽北赛道，力争创造更好成绩。

株洲国际赛道

成为最速四门量产车，圈速成绩为1分41秒806。这一成绩打破了多项国内赛道纪录，展现了小米在高性能电动车领域的实力。

上海国际赛车场

以2分9秒944的圈速，超过保时捷Taycan Turbo GT，成为上海国际赛车场最速量产车。这一成绩证明了小米SU7 Ultra在专业赛道上的卓越表现。

极速测试

在盐城试验场完成极速实测，达到了359.71km/h的惊人速度，超过官方宣称的350km/h最高时速。
